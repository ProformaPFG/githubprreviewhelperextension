🚀 How It Works

  1. Navigate to any GitHub PR's Files tab
  2. The extension automatically extracts and analyzes code changes
  3. Issues are flagged with color-coded severity levels
  4. Hover over issues to see detailed remediation guidance
  5. Use the popup to export findings as a markdown comment
